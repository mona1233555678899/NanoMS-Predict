# NanoMS-Predict: Enterprise R&D Platform

An AI-powered In-silico simulation platform powered by XGBoost Regressor to predict nanoparticle apparent permeability (Papp) across the Blood-Brain Barrier (BBB) and evaluate neuro-cytotoxicity safety profiles during early-stage drug design.

---

## Core Features & Architecture
* Advanced Predictive Modeling: Utilizes dual-optimized XGBoost models trained on key physicochemical parameters.
* Industrial Screening: Real-time multi-parametric evaluation for drug delivery optimization.
* Clean & Modern UI: Interactive dashboard built using Gradio (Monochrome Theme).
